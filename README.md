# Proyecto Final

## Code & Data Bootcamp: Git, Bash y R

#### Samuel Márquez, Emiliano Almaráz, Gerónimo Chávez, Emiliano Sánchez

El proyecto consiste en un análisis exploratorio de los datos sobre la Copa Mundial de la FIFA.

Los datos utilizados están disponibles en TidyTuesday: [Link al repositorio en Github](https://github.com/rfordatascience/tidytuesday/tree/main/data/2022/2022-11-29), y aparecen dentro de la carpeta data con el nombre "wcmatches.csv".

Este proyecto se realizó en R, y los resultados finales se encuentran en el archivo [ProyectoFinal.qmd](ProyectoFinal.qmd), mismo que tambíen se puede visualizar como una [página web](ProyectoFinal.html).

Dentro del repositorio hay 4 carpetas:

- [archive](archive/): Esta carpeta contiene un análisis exploratorio inicial en la forma de un documento de quarto y un archivo .R. Adicionalmente contiene algunas de las pruebas iniciales del equipo en la creación de gráficas.

- [ProyectoFinal_files](ProyectoFinal_files/): Contien otras dos carpetas - (libs)[libs/] contiene algunas de las librerías utilizadas. - (figure-html/) contiene algunas de las figuras desarrolladas durante la exploración y experimentación con los datos.

- [data](data/): Contiene el archivo *wcmatches.csv* utilizado durante el proyecto.

- [figures](figures/): Contiene una imagen decorativa.
